JavaScript 教程

基本简介

JavaScript 是互联网上最流行的脚本语言，是一种轻量级编程语言，可插入 HTML 页面并由浏览器执行。

JavaScript 和 Java 无论是从概念上还是设计上都是两种完全不同的语言。

JavaScript 由 Brendan Eich 发明。它于 1995 年出现在 Netscape 浏览器中，并于 1997 年被欧洲电脑制造商协会 ECMA 采纳。

相应的 JavaScript 标准列举如下：

年份 名称 描述
1997 ECMAScript 1 第一个版本
1998 ECMAScript 2 版本变更
1999 ECMAScript 3 添加正则表达式、try/catch
ECMAScript 4 没有发布
2009 ECMAScript 5 添加 strict mode 以及 JSON 支持
2011 ECMAScript 5.1 版本变更
2015 ECMAScript 6 添加类和模块
2016 ECMAScript 7 添加指数运算符、Array.prototype.includes

JavaScript 的主要用途

  • 直接写入 HTML 输出流
  • 对事件的反应
  • 改变 HTML 的内容
  • 改变 HTML 的图像
  • 改变 HTML 样式
  • 验证输入

基本用法

HTML 中的脚本必须位于 <script></script> 之间，脚本可以放置在 <head> 部分、<body> 部分、外部文件 .js 中。

执行方式：

  1. 检查>开发者工具>Console 窗口>输入代码并回车执行
  2. Sources 面板>Snippets 选项卡>Create new snippet>输入代码并保存执行

输出方式

JavaScript 没有打印/输出函数，但可以通过不同方式进行输出：

  1. 弹出警告框
    使用 window.alert()
  2. 插入到 HTML 元素
    document.getElementById(id).innerHTML = ""
    根据 ID 属性查找 HTML 元素并修改内容
  3. 写入到 HTML 文档
    使用 document.write()
  4. 写入到控制台
    使用 console.log() 在浏览器中显示值

语法

字面量 (一般固定值)

数字 (Number) 字面量：整数/小数/科学计数；
字符串 (String) 字面量：使用单引号/双引号；
表达式字面量：用于计算
数组 (Array) 字面量：定义数组 []
对象 (Object) 字面量：定义对象 {key:value}
函数 (Function) 字面量：定义函数 function name(a,b){return a+b;}

变量 (存储数据值)

变量名必须以字母开头；
变量名可以用 $ 和 _ 开头，但不推荐；
变量名对大小写敏感
定义并赋值：var a = 5
可以在一条语句中声明多个变量，以逗号分隔，可跨多行，不可同时赋一个值。
如 var x,y,z = 1; 结果为 x,y undefined; z=1
重新声明变量，值不会丢失。

2015 年前使用 var 进行声明，2015 年后 ES6 可以使用 const 定义常量，let 定义限定范围内作用域的变量。Link

操作符

  • 算数运算符：计算值
  • 赋值运算符：给变量赋值
  • 位运算符
  • 条件运算符
  • 比较运算符
  • 逻辑运算符

其他

JavaScript 对大小写敏感；
JavaScript 使用 Unicode 字符集；
JavaScript 会忽略多余的空格；
可以在字符串中使用反斜杠 \ 进行换行；
注释符号：单行 // 多行 /*...*/

语句

分号：分隔 JavaScript 语句。
语句标识符：
break 用于跳出循环。
catch 语句块，在 try 语句块执行出错时执行 catch 语句块。
continue 跳过循环中的一个迭代。
do … while 执行一个语句块，在条件语句为 true 时继续执行该语句块。
for 在条件语句为 true 时，可以将代码块执行指定的次数。
for … in 用于遍历数组或者对象的属性（对数组或者对象的属性进行循环操作）。
function 定义一个函数。
if … else 用于基于不同的条件来执行不同的动作。
return 退出函数。
switch 用于基于不同的条件来执行不同的动作。
throw 抛出（生成）错误 。
try 实现错误处理，与 catch 一同使用。
var 声明一个变量。
while 当条件语句为 true 时，执行语句块。
